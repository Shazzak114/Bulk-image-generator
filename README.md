# 🚀 Advanced AI Image Generator Pro

A powerful desktop application for generating bulk AI images with advanced features and intuitive controls.

---

## 📌 What This Tool Does

This is a full-featured image generation application with a modern, user-friendly interface.  
Built with **Python + Tkinter**, it allows you to generate multiple AI images at once using simple text prompts, with extensive customization options.

---

## ✨ Key Features

### 🎯 Core Generation

- **Batch Generation**  
  Write multiple prompts (separated by blank lines) and generate all at once  

- **Free & Fast**  
  Uses the **Pollinations.AI API** with support for the **Flux model**  

- **Multiple Sizes**  
  Generate images from `512x512` up to `1920x1080`, including mobile formats  

- **Preview Canvas**  
  Instantly preview generated images inside the app  

---
